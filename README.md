# 📲 Sentiric Telecom Client Sdk

Mobil, masaüstü ve CLI istemcileri için 'Softphone Beyni' olarak çalışan Rust tabanlı SIP/RTP motorudur.

## 🚀 Hızlı Başlangıç
```bash
cargo build
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **Anayasal Konum:** [sentiric-spec/spec/libraries/telecom-client-sdk.yaml](https://github.com/sentiric/sentiric-spec)
